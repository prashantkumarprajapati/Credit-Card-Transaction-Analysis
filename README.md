💳 Credit Card Transaction Analysis

📌 Project Overview

This project focuses on analyzing large-scale credit card transaction data to uncover spending patterns, detect anomalies, and identify customer segments. Using Python-based data analysis techniques, the project delivers meaningful insights to support financial decision-making.

🚀 Key Features
  🔍 Analysis of 50,000+ transactions
  📊 Identification of spending behavior & trends
  ⚠️ Detection of anomalies and outliers
  👥 Customer segmentation based on transaction patterns
  📈 Data visualization for actionable insights

  
🛠️ Tech Stack
